# Sistema de Gestión de Reservas Internas – Empresa123Messi

Proyecto realizado para **Taller de Proyectos II**.

Incluye:
- Selección de salas
- Calendario básico
- Validación de disponibilidad
- Registro de reservas
- Estructura organizada con buenas prácticas
- Uso de Git para control de versiones

## Cómo ejecutar
Abrir el archivo `index.html` en cualquier navegador.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- JSON simulado como base de datos

## Estado del proyecto
Funciones principales implementadas
